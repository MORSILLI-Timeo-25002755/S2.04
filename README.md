# Générateur de jeu de donnée pour une BDD (S2.04)
## Installer Python
```bash
sudo apt install python
```
## Lancer un environnement virtuel Python

```bash
python3 -m venv .venv
source .venv/bin/activate
```

## Installer les dépendances

```bash
pip install -r requirement.txt
```

## Lancer le script
```bash
python generate.py
```

> Attention ! Il faut mettre le fichier `intention.sql` au même niveau que le script `generate.py` !
